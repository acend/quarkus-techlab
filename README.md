# Quarkus Techlab

Techlab for writing Java microservices implemented in Quarkus

## Content

  * Introduction to Quarkus
  * REST API Example
  * Run, Build, Deploy
  * Messaging with Quarkus
  * Monitoring 

## Prerequisites

To follow along this techlab you will need: 
  * Java OpenJDK 11
  * GraalVM CE 19.3.1
  * Docker
